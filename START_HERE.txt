🚀 3 ÉTAPES FINALES (15 minutes)
═══════════════════════════════════════════════════════════════════════

ÉTAPE 1️⃣  : Créer un projet Firebase
   → Allez sur: https://console.firebase.google.com
   → Cliquez "Ajouter un projet"
   → Entrez un nom: "global-files"
   ⏱️  Temps: 5 minutes

ÉTAPE 2️⃣  : Récupérer identifiants + créer .env
   → Dans Firebase: Paramètres > Vos applications > Web
   → Copiez la configuration firebaseConfig
   → Créez le fichier .env à la racine du projet
   → Collez vos identifiants (voir .env.example)
   ⏱️  Temps: 5 minutes

ÉTAPE 3️⃣  : Activer Firestore + Tester
   → Firebase: Firestore Database > Créer une base
   → Mode: "Démarrer en mode test"
   → Région: "europe-west1"
   → Lancer: npm run dev
   → Tester: Importer des données
   → Vérifier: Firestore Console > Collection "fichierGlobal"
   ⏱️  Temps: 5 minutes
